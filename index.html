Porquê este projeto?

🚌 Necessidade

Apesar de bastante desenvolvido, o sistema de transporte público em Lisboa apresenta bastantes falhas a nível de acessibilidade.

📖 Pedidos

De acordo com feedback de terceiros, este projeto é uma mais valia para toda a comunidade que sofra de deficiência visual no distrito de Lisboa.

🌍 Impacto

A acessibilidade é um direito de cada cidadão, independentemente das suas limitações físicas.
